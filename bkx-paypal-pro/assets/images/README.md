# PayPal Pro Images

## Required Images

Place the following images in this directory:

1. **paypal-logo.svg** - PayPal logo for the payment gateway icon
   - Dimensions: 60x38px or similar aspect ratio
   - Format: SVG (vector)
   - Download from: PayPal Brand Guidelines

## PayPal Brand Assets

Official PayPal logos and brand assets can be downloaded from:
https://www.paypal.com/us/webapps/mpp/logo-center

## Usage

The PayPal logo is referenced in:
- `src/Gateway/PayPalGateway.php` (line 38)
- Used as the payment method icon in checkout

# DirectPay EU Add-on - Development Documentation

## 1. Overview

**Add-on Name:** DirectPay EU
**Price:** $59
**Category:** Payment Gateways

### Description
European payment gateway with GDPR compliance, multi-currency support, and localized payment methods.

---

## 2. Key Features

- SEPA Direct Debit
- Credit/debit cards (European focus)
- Local payment methods (iDEAL, Sofort, Giropay)
- Multi-currency (EUR, GBP, CHF, etc.)
- GDPR compliant data handling
- PSD2/SCA compliance
- Recurring SEPA mandates

---

## 3. Technical Stack

- DirectPay REST API
- SEPA XML format
- Strong Customer Authentication (SCA)

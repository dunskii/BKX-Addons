# PayMaya Add-on - Development Documentation

## 1. Overview

**Add-on Name:** PayMaya
**Price:** $49
**Category:** Payment Gateways

### Description
Philippines-based payment gateway with comprehensive API integration and local payment method support.

---

## 2. Key Features

- Credit/debit card processing
- PayMaya wallet payments
- Installment payments
- Recurring billing
- Tokenization for saved cards
- Multi-currency support (PHP, USD)

---

## 3. Technical Stack

- PayMaya Payments API v1
- Vault API for tokenization
- Webhook events
